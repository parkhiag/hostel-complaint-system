# 🏨 Hostel Complaint Tracking System

A web-based platform to streamline complaint management in hostels, replacing inefficient manual systems with a transparent and real-time solution.

## 🚀 Features

- Complaint submission by students
- Real-time complaint tracking
- Admin dashboard for complaint management
- Role-based authentication
- Responsive UI (Mobile + Desktop)

## 🛠 Tech Stack

- React.js
- Vite
- Supabase (Authentication + Database)
- JavaScript
- CSS
- Vercel (Deployment)

## 📦 Run Locally

```bash
git clone https://github.com/YOUR-USERNAME/hostel-complaint-system.git
cd hostel-complaint-system
npm install
npm run dev
